Wow I have always wanted to use python but I have been learning so many
helpful things in R! Why don't we use both with reticulate? I can use R
to assign something.

\begin{Shaded}

\end{Highlighting}
\end{Shaded}

Then I can use python to assign something else.

\begin{Shaded}
\begin{Highlighting}[]

\end{Highlighting}
